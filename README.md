## 环境要求
JDK8+


## maven   
当前snapshot版本：**4.7.2-SNAPSHOT**   

若使用snapshot版本，请添加snapshotRepository仓储：   
```xml   
<repository>
     <id>oss</id>
     <url>https://oss.sonatype.org/content/repositories/snapshots/</url>
    <snapshots>
        <enabled>true</enabled>
    </snapshots>
</repository>   
```
